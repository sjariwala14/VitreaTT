Deployment Architecture Options

Two deployment models have been evaluated for consolidating VitreaAdvanced across Mount Elizabeth Hospital and Mount Elizabeth Novena.

Option Architecture Advantages Considerations
Option 1 (Recommended) Single Database Centralized
  • • One central VIMS server & database
  • • Streamlined management & PACS integration
  • • Enterprise-wide data access
  • • Single point of maintenance
  • • Doctors from both sites can access all patient data
  • • Requires cross-domain connectivity setup
Option 2 Separate Databases
  • • Two independent VIMS servers
  • • Physically segregates patient data by site
  • • Each site is self-contained
  • • More complex to manage
  • • Less efficient resource utilization
  • • Limited cross-site access

Important Note: VitreaAdvanced can work across multiple domains. Separate domains at each site are not an issue as long as communication between them is permitted by Parkway Radiology's IT department.

Implementation Requirements

  • Hardware Refresh: Replace 4 aging workstation PCs at Mount Elizabeth Hospital (6+ years old) with new low-cost PCs capable of running VitreaAdvanced.
  • Server Infrastructure: Rack space required at Mount Elizabeth Hospital data center for new hardware. UPS recommended for deployment.
  • LungCAD Migration: Obsolete LungCAD server can be scrapped. LungCAD functionality is integrated within VitreaAdvanced without separate server requirement.
  • Network Configuration: Ensure cross-domain connectivity between hospital sites. VitreaAdvanced supports multiple domains.
  • PACS Integration: Integration with Infinitt PACS using 'external link icon' for seamless launch from PACS worklist.

Recommendation & Benefits

Option 1 (Single Database Centralized) is strongly recommended for Parkway Singapore. This approach maximizes the benefits of consolidation while leveraging the high-speed WAN between sites.

Moving the existing VitreaAdvanced instance from Novena to the Mount Elizabeth Hospital data center creates a unified enterprise platform with centralized management, simplified PACS integration, and ready access to patient data from any site.

Redundancy & High Availability

VitreaAdvanced supports fully redundant deployments through load balancing and VMware High Availability capabilities. No license charge for additional VIMS nodes in existing deployment.

Virtualization Ready

100% virtualized solution certified "VMware Ready". All components including VitreaCore and Advanced 3D rendering servers can be virtualized with 100% CPU rendering (no GPU required).

System Longevity

Software-only solution allows you to maintain control of your environment. Upgrade through purchase of standard servers with off-the-shelf hardware or in virtualized environment.
