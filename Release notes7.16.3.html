📘 Introduction

Covers Vitrea Advanced Visualization 7.16.0, 7.16.0.SU01–SU04, 7.16.1, 7.16.2, 7.16.2.SU01–SU02, 7.16.3, 7.16.3.SU01, and 7.16.3.SU02. Includes new features, enhancements, cautions, deprecated functionality, resolved and known issues.

⚠️ CAUTION: For general Safety and Regulatory Considerations, refer to the Education and Reference Guide. Federal law restricts this device to sale by or on order of a physician (21 CFR 801.109(b)(1)).
📘 Intended Use Statements are available in the About Vitrea Advanced Visualization Guide.

✨ New Features & Enhancements (by version)

🔹 7.16.3.SU02
  • VSTP Language Support: Slovakia, Greece, France, Norway (VSTP1 v8.10 SP0010, VSTP2 v2.2 SP0001).
  • Slovakian & Greek translations: About Vitrea, General ERG, Multi Modality Viewer, MR Stitching, 3DMPR, PET/CT Viewer guides.
🔹 7.16.3.SU01
  • Brain Perfusion 4D scanner support: Helical datasets from single-frame Aquilion Serve SP, NoMotion from multi-frame Aquilion Prime.
  • Windows Server 2025 support for Extend deployments.
  • Medis Suite updates: QMass Function (automated LV/RV by deep learning, DSI, T2w, T1/T2*, iron overload from T2*), QFlow 4D (auto mitral valve tracking, noise removal).
  • VSTP2 updates: Auto MPR Spine (new – spine labeling), Spectral Composition Analysis (2Rot DE scan mode for Gout, remove small compositions), bug fixes for Auto MPR Brain, Subtraction Viewer, Spectral Stone Analysis, Spectral Analysis. (* not available in US).
🔹 7.16.3
  • 16-bit single-frame SC curved inset batches (window/level in external DICOM viewers).
  • Carotid Analysis JIT Processing – automatic vessel probing, editable results.
  • External App Launcher – launch single study into external app with separate datastore.
  • Improved JiT Task Scheduling & CT Bone Removal for 1K studies – faster queue, better slot utilization, 1K support.
  • Certificate visibility (System Health, logs, expiry warnings).
  • SQL Server 2022 support (Standard/Express).
  • Mirada XD 4.2 – PET hotspot finder, thick slab, 2D MPR rotation, new review modes, dark UI, multi-destination export, UI/IFU translations.
🔹 7.16.2
  • CT Vitrea Colon CAD – secondary reader for polyps (alternative to iCAD; not for sale in US).
  • Cylinder Tool – 2D/3D measurements (excl. CFA, Myo, flythrough).
  • Lung Lobe Segmentation – CT Lung Lobe Density Analysis (HU density/volume per lobe; not US).
  • CT TAVR Planning Assist – auto valve plane, annulus, coronary distances; not US.
  • DICOM VR types UC/UR for Siemens Multi-energy CT.
  • Open Rib auto-labeling (user guide updated; not US).
  • AP clinical apps launch from Study List/Plus Tab + Single Sign-On for AP results.
  • VSTP2 additions: Spectral Analysis, Composition, Stone Analysis; Lithuanian/Bulgarian language.
  • Multi Modality Viewer: Hanging Layout, Oblique Batch MPR, reset views.
  • Olea Sphere SP37 – Java 17, cybersecurity updates, Breastscape Portuguese/Romanian, Functional MR translations, auto-save results.
  • Cybersecurity: Guest accounts disabled by default.
🔹 7.16.1
  • Automated user preset transition from 7.16.0.
  • Brain CTP+ segmentation improvements (ventricles, CSF).
  • Solution Health Reporting – added "numAssigned" field.
  • Open Rib 1.2.3.8 (functionality gaps closed).
  • "What's New" menu, MyVitrea portal in Help Central.
  • 4‑user Enterprise Single Server hardware spec.
  • .NET 6 → .NET 8 upgrade.
  • Conduit quit verb configurable (close/quit/hide).
  • Windows Server 2012 R2 support retired, repair button removed.
🔹 7.16.0
  • Perivascular CT Density: batch rotation cover page, histogram persist.
  • AV AP Integration: filters >512px, configurable display names.
  • Application Platform: session timeout, CORS fixes, OTS updates.
  • External Integration (Conduit) persists last used monitor.
  • Nvidia driver benchmarking (2024 Q1).
  • DICOM SCP supports JPEG-LS transfer syntax (.80).

🔒 Known Limitations

  • 7.16.3.SU01: Open Rib JIT algorithm does not support Enhanced CT Image Storage (SOP Class 1.2.840.10008.5.1.4.1.1.2.1). Manual support available.
  • 7.16.0: Multiple Monitor configurations supported only on two homogeneous monitors with identical resolution, DPI, orientation.

🗑️ Deprecated Functionality

7.16.0: Windows Server 2012 R2 support retired (Microsoft EOL October 2023). No longer installed.
