User Interface Updates

Vitrea Advanced Visualization has a refreshed user interface for the Study List and +Tab with visual changes to align with other Canon Medical products.

Modern Design

Study List and +Tab now display with new colors and fonts for improved readability and aesthetics.

Enhanced Icons

Updated status icons and pop-ups for better user experience and clearer visual communication.

Improved Menus

Updated dialog and context menus including Edit Patient, Anonymize Patient, Export Dialog, and Study List Settings.

Clinical Enhancements

New features and improvements to clinical workflows across multiple modalities.

CT Lung Lobe Density Analysis 7.16.2

Leverages deep learning to measure HU density and volume in various lung lobes with precise lobular level breakdown.

CT TAVR Planning Assist 7.16.2

Automatically initializes valve plane, coronary distances, aortic angle and annulus measurements using deep learning.

Carotid & Vertebral Vessel Workflow

Improved segmentation algorithm for better vessel initializations with vertebral arteries extending closer to basilar bifurcation and aortic arch.

DICOM SR Output

Generate DICOM SR output from CT VScore application including calcium scoring guidelines, slice thickness, and density threshold.

Integration & Security Enhancements

Improvements to system integration, workflow, and security features.

Multi-Monitor Persistence

The monitor selection for external integrated launches is now persistent, maintaining multi-monitor layout consistency.

User Presets Transfer

Easier transition of modified user presets from previous versions via automation during upgrades.

Security Improvements

Updated TLS configuration, NVIDIA driver security fixes, and architectural improvements for better data security.

DICOM Enhancements

Support for JPEG-LS transfer syntax and improvements to Retrieved DICOM Lifecycle Management.
