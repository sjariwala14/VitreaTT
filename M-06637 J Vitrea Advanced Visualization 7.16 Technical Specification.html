🧩 Virtualization Specifications (VMware vSphere®)

Hypervisor: VMware vSphere® 8.0+ with VMware Tools™ 11.3+, VMXNET3 drivers. Storage must meet IOPS/throughput criteria.

📐 512x512 Matrix Support

Vitrea Management Server 18,000 MHz (up to 30 users) · 24GB RAM · 250GB OS VMDK + 50GB DICOM receive VMDK · Storage ≥1500 IOPS, 80 MB/sec throughput <15ms latency · Network 1 Gb/s
SQL Server 5900 MHz (multi-core vCPUs) · 4GB RAM · 250GB OS VMDK + 40GB SQL VMDK · SQL Server 2022/2019 Standard recommended
Vitrea Application Server (GPU) 19,000 MHz per user · 24GB RAM · 128MB video memory · vGPU “Q” profile with ≥4GB framebuffer (e.g., T4-4Q) · ≥900 CUDA cores per user

📐 1024x1024 Matrix Support (“1K”)

Vitrea Management Server 25,000 MHz (up to 30 users) · 48GB RAM · 250GB OS VMDK + 100GB DICOM receive VMDK · Storage ≥1500 IOPS, 175 MB/sec throughput <15ms latency · Network 10 Gb/s · NAS must provide ≥300 MB/sec R/W
SQL Server 5900 MHz (minimum 4 cores) · 4GB RAM · 250GB OS VMDK + 40GB SQL VMDK · SQL 2022/2019 Standard · Network 10 Gb/s
Vitrea Application Server (vGPU) 25,000 MHz per user · 28GB RAM + 16GB page file · 256MB video memory · vGPU “Q” profile with ≥8GB (12GB recommended) framebuffer · ≥1280 CUDA cores per user
CPU Calculation: Required MHz ÷ CPU speed (MHz) = vCPUs (round up). Example: 18,000 MHz ÷ 2400 MHz = 7.5 → 8 vCPUs. Assign all vCPUs to one processor in VMware.

Multi‑Modality Server

512x512 Matrix Up to 4 concurrent users, 8000 slices · 19,000 MHz · 24GB RAM · 16MB video memory · 1 Gb/s network
1024x1024 Matrix (GPU Rendering) Up to 4 users, 8000 slices · 25,000 MHz · 28GB RAM + 16GB page file · 256MB video memory · 10 Gb network · vGPU acceleration options
Caution: Canon considers these virtualization specs as minimum for up to 22 Vitrea Application Servers with two Multi‑Modality Servers. Varied workloads may require adjustments. Consult Canon technical contact.

🎛️ NVIDIA® Data Center Products & vGPU Requirements

Important: Custom hardware or virtualized deployments require NVIDIA GPU appropriate for environment. vDGA or vGPU integration required. Review vendor documentation. Canon provides only Vitrea AV‑specific requirements.

GPU Selection Guidance: Choose GPU supported by hardware, OS, and virtualization vendors. Must support WDDM driver mode. Data center GPUs (L4, L40S) require NVIDIA RTX Virtual Workstation (vWS) licenses. vGPU Manager should be set to “Fixed share” (recommended) or “Equal share”.
