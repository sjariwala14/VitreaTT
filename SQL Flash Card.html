Guide Summary

The "VPMC-17204E SQL Server Setup Guide" provides comprehensive instructions for setting up and upgrading SQL Server Standard Edition for Vitrea® Advanced Visualization software versions 7.16.2.SU01 and later.

Key Sections:

  • Prerequisites for Installation: Complete Active Directory and Operating System setup guides, have access to specific SQL Server installation media (SQL Server 2016 SP2, 2017, or 2019 CU5+).
  • IT Roles: Defines responsibilities for Database Administrator, Domain Administrator, IT Administrator, Network Administrator, and VMware Administrator.
  • Installation Procedures: Detailed steps for installing a new SQL Server instance, including prerequisites like .NET Framework 4.5+ and SQL Server Management Studio (SSMS).
  • Configuration: Only Named Instances (e.g., VITREASQLSTD) are supported for local installations. Configuring SQL Server services (SQL Server Agent to Manual, Database Engine and Browser to Automatic).
  • Network Protocols: Disable Shared Memory and Named Pipes, set specific TCP port (e.g., 2222) for IPAll.
  • Security: Enabling TLS/SSL encryption, managing certificate permissions, registering SPNs for Kerberos.
  • Performance: Limit SQL Server memory to 4GB, disable CEIP service, optionally change TempDB location.
  • Permissions: Modify sysadmin permissions to enforce "least privilege" by assigning specific db_ roles.
  • Upgrade Procedures: Steps for upgrading from SQL Server 2014 Standard Edition to newer versions (2016 SP2, 2017, or 2019 CU5+).

Important Notes:

  • Microsoft ends support for SQL Server 2014 on July 9th, 2024.
  • This guide is exclusively for Enterprise deployments (workstation/extend deployments install SQL Server automatically).
  • After SQL Server setup/upgrade, proceed with Management Server Installation Guide.
