‚ùó Common Errors & Fixes
Applications are not launching
  • Check licensing on: http://localhost:8700/management/api/license/usage
  • GPU resources in use ‚Äì ensure other sessions closed
ERR_DNS_FAIL 
Open Internet Options > Connections > LAN settings
Ensure "Bypass proxy for local addresses" is checked.
Archive CD/DVD IMAPI Error
  • Check DVD hardware
  • Uninstall virtual DVD drives
StudyList displays no data
  • Check UI filters (set Study Date = Any)
  • Verify VAT data directory & AutoDelete settings
