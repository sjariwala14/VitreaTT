New Features and Enhancements

The following features, functionality, and enhancements have been incorporated into Vitrea Advanced Visualization 7.16.0, 7.16.0.SU01, 7.16.0.SU02, 7.16.0.SU03, 7.16.0.SU04, 7.16.1, 7.16.2, 7.16.2.SU01, and 7.16.2.SU02 releases, addressing 41 issues across clinical, system, and administrative categories.

CT Vitrea Colon CAD 7.16.2

Vitrea software now has the capability to use the Vitrea Colon CAD algorithm as a secondary reader for finding polyps. This is an alternative to iCAD Colab CAD, which is still an option.

NOTE: Canon Medical System Corporation Colon CAD application is not available for sale in the United States.

Lung Lobe Segmentation 7.16.2

CT Lung Lobe Density Analysis is a new application/preset that leverages deep learning technology, extending the Lung Density Analysis application to measure HU density and volume in the various lung lobes. It segments the right lung into upper, middle, and lower lobes and the left lung into upper and lower lobes, assigning HU density, percentile, and volume measurements to each. The lobular level breakdown delivers precise measurements and reporting of density and volume for a more informative lung density analysis workflow.

NOTE: CT Lung Lobe Density is not available for sale in the United States.

CT TAVR Planning Assist 7.16.2

CT TAVR Planning Assist is a new application/preset that leverages deep learning technology to automatically initialize the valve plane, the distances of the coronaries to the valve plane, aortic angle, and annulus measurements. It streamlines and accelerates the TAVR planning workflow, while also helping reduce the variability among users with different experience levels. Includes configurable display names in the Algorithm Processing Tab.

NOTE: CT TAVR Planning Assist is a separately licensable option and is not available for sale in the United States.

Cylinder Tool 7.16.2

The Cylinder tool provides 2D/3D visualization and measurements of a virtual cylinder object in relation to other anatomies. The tool is a generic functionality that is available across all applications and workflows except Cardiac Functional Analysis, Myocardial Perfusion, and flythrough protocols. Users can create and manipulate the cylinder in MPR as well as 3D.

Vitrea Brain CTP+ Enhancements 7.16.1

In rare situations, under-segmentation of brain ventricles especially for patients with large size ventricles, under-segmentation of CSF due to beam hardening, or over-segmentation of brain due to low quality scan acquisition may occur. These segmentation issues have been addressed and improved.

Perivascular CT Density 7.16.0

The following usability improvements have been made to the Perivascular CT Density application: Batch or movie rotation includes cover page that contains color key for Perivascular and SurePlaque regions; Perivascular CT Density histogram dialog size and location saved.

VSTP2 Applications 7.16.2

In Vitrea AV 7.16.2, the following VSTP2 applications were updated: Spectral Analysis application added; Spectral Composition Analysis application added; Spectral Stone Analysis application added. Additional minor enhancements and some bug fixes were completed on the other VSTP2 applications.

Open Rib Updates 7.16.1, 7.16.0.SU02

In Vitrea 7.16.1 and 7.16.0.SU02, the Open Rib application was updated to close some functionality gaps. Version 1.2.3.8 includes defect fixes particularly focused on Auto Export, improving rib fracture examination workflows. Auto-Labeling for Open Rib has been updated in the User Guide and is available for use. OpenRibPlugin remains at version 1.1.9.

NOTE: Labeling of ribs is not available for sale in the United States.

Medis® Suite Updates 7.16.0

Medis Suite 4.0.62.4 has been updated for Vitrea AV 7.16.0. QFlow, QFlow4D, QMass, and QStrain are no longer delivered as individual applications, but are encompassed within the MedisSuite MRCT product. Release Notes for this application can be found on Help Central.

Enhancement of Clinical User Experience 7.16.1

Enables easier transition of modified user presets from 7.16.0 to 7.16.1 via automation during upgrades, enhancing clinical user experience (VTL-10938).

Enterprise CT 1K Support 7.16.2.SU01, 7.16.0

Information has been added to our documentation to clarify additional hardware and configuration steps necessary to support 1K (1024x1024) data on an Enterprise deployment. Additional minor enhancements were made to appropriately manage handling of incoming 1K studies.

Supporting New DICOM Value Representation Types 7.16.2

Additional Value Representation (VR) types are supported as of 7.16.2 to enable ingestion of DICOM data containing the Multi-energy CT Image Module for the Siemens scanner. Specifically, the UC and UR VR types were added to support (FRDB-6508).

Multi Modality Viewer Updates 7.16.2

Multi Modality Viewer application updates include new features of Hanging Layout, Oblique Batch MPR, and the ability to reset views to a previous layout.

Cautions and Notes

Please review the following important considerations before deploying or using this release.

Mirada NMX Availability (7.16.2, All Deployments)

The Mirada NMX application is no longer available in the EU for new customers.

Large Dataset Processing (7.16.0, All Deployments)

Vitrea may not be able to process large (with regard to matrix size and/or number of slices) datasets in some circumstances. Contact your system administrator or Canon Customer Support for more information.

Auto MPR Operation Manual Updates (7.15.8, 7.16.0, All Deployments)

For the Operation Manual for Auto MPR, the following updates should have been included:

  • On page 14, change the series description from "Brain Scan" to "Head Scan".
  • On page 18, update the "Terminating the application" subsection to 4.2.5.

.NET 6.0 Runtimes (7.15.7, All Deployments)

.NET 6.0 runtimes for .NET, ASP.NET Core, and .NET Desktop are installed automatically by the InstallVitrea.exe file as a part of the software installation process. These runtimes should be updated regularly with up-to-date security patches and bug fixes. Contact Canon Customer Support for more information about updating .NET runtimes. While .NET 6.0 is a Microsoft product, it is distinct from .NET Framework 4.8. .NET Framework 4.8 is a component of the Windows Operating System whereas .NET 6.0 is an independent product. These products have different support policies. Contact Canon Customer Support for more details.

Multi Modality Viewer Measurement Units (7.15.7, All Deployments)

In the Multi Modality Viewer Education and Reference Guide, the following NOTE belongs in the "Understanding Measurement Values and Units" section:

For a DX image to show "mm", it needs:

  • Tag (0018,1164) ImagerPixelSpacing
  • Tag (0018,1114) EstimatedRadiographicMagnificationFactor

This will show "mm ERMF" or:

  • Tag (0028,0030) PixelSpacing
  • SOP class UID of "X-Ray 3D Angiographic Image Storage" - 1.2.840.10008.5.1.4.1.1.13.1.1

This will show "mm".

The dataset in question has no EstimatedRadiographicMagnificationFactor tag, so it cannot show "mm ERMF". And the SOP class UID is "Digital X-Ray Image Storage - For Presentation", so it cannot show "mm". Thus it falls back to "pixels".

Windows Operating System Support (All Versions, All Deployments)

Canon Medical strongly recommends that Vitrea AV software be installed on currently Microsoft-supported Windows Operating Systems. We can not guarantee that the Vitrea AV Workstation or Vitrea AV clients for multi-user systems like Extend and Enterprise will continue to function properly on Windows Operating Systems that Microsoft no longer supports. Contact Canon Medical Customer Support for more information.
