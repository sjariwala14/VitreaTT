üíæ Database Backup & Restore

Scheduled Backup:

Backups run daily at 1:00 a.m. local time and retain 5 copies. 
Default paths:
Enterprise: \\[machinename]\nasroot\DBBackup
Single Server/Workstation: E:\VitreaData\DBBackup

Manual Backup in SQL:

Connect with SQL Server Management Studio
Right-click VIDicom > Tasks > Back Up
Set Backup type = Full
Choose destination path with .bak extension

Manual Restore:

Stop all services:
wmic service where "name like 'Vitrea.%%'" stopservice
wmic service where "name like 'VSP%%'" stopservice
üóÑÔ∏è Storage Management

Change Storage Location:

  1. Stop all services.
  2. Detach VIDicom database in SQL.
  3. Copy data to new NAS path.
  4. Update Registry keys: 
    HKLM\SOFTWARE\Vital Images\VIMS > NasRoot
HKLM\SOFTWARE\Vital Images\Vitrea > VitreaDataRoot
  5. Update configuration files: 
    C:\ProgramData\Vital Images\Application\conf\vital-config.xml
  6. Update IIS paths.
  7. Restart machine.
