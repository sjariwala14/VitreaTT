What is the recommended number of network connections per server?

Two 1GB connections (one for Out-of-Band, optional third for iLO).

What VLAN requirement exists for multi-server deployments?

All servers must be in the same VLAN (no mix of public/private VLANs).

What is the recommended RAID configuration for an Enterprise Single Server?

RAID 10 with four 960GB SSDs.

What is the minimum CUDA cores/user requirement for a 512x512 vGPU profile?

900 CUDA cores per user.

What SQL versions are supported for Vitrea 7.16?

SQL Server 2016 SP2, 2017, or 2019 CU5+.

Why is CORS enforcement critical starting with Vitrea 7.15?

It blocks communication if servers are in mixed public/private VLANs.

Issue: After a power outage, the server boots to a BitLocker recovery screen.

Cause: TPM 2.0 didn't release the key during abrupt shutdown.

Fix: Enter the 48-digit recovery key, then suspend/resume BitLocker via PowerShell:

Suspend-BitLocker -MountPoint "C:"
Restart-Computer
Resume-BitLocker -MountPoint "C:"

Issue: Application Servers fail to join the domain with "DNS name does not exist."

Cause: Missing forward/reverse DNS records.

Fix: Verify A and PTR records exist in DNS:

nslookup <hostname>
nslookup <IP>

Issue: VitreaConduitService fails to start (Port 443 in use).

Cause: IIS or another service bound to 443.

Fix: Identify the conflicting process:

netstat -ano | findstr :443

Reconfigure VitreaConduitService or stop the conflicting service.

Issue: SQL backups fail with "Cannot open backup device."

Cause: Vitrea-SQL$ lacks write access to \\NAS\DBBackup.

Fix: Grant Change permissions to the share and verify SQL Agent runs under Vitrea-SQL$.

Quick Tips & Summary

VLANs

  • Uniform VLAN required
  • No mixing of public/private VLANs
  • CORS blocks mixed-VLAN communication in Vitrea 7.15+

GPUs

  • Use "Q" profiles (e.g., t4-4q)
  • Minimum 900 CUDA cores/user
  • 8GB framebuffer for 1024x1024 datasets
  • Perpetual licenses recommended

SQL Server

  • Supported versions: 2016 SP2, 2017, 2019 CU5+
  • Default port: 2222 for named instances
  • Exclude .mdf/.ldf from antivirus scans
  • Vitrea-SQL$ needs write access to backup share

Security

  • Enable BitLocker for C: and E: drives
  • Store recovery keys securely off-server
  • Required firewall ports: TCP 443, 3002, 3389
  • Disable SMBv1

Hardware

  • Management Server: RAID 1 (OS) + RAID 10 (data)
  • Application Server: RAID 1 (SSD or HDD)
  • Set power plan to "High Performance Compute"
  • Disable NUMA on Dell servers

Key Commands

Hardening Script:

cd "E:\Vitrea_Resources\Utilities\OS Configuration Scripts\Windows Server\Post"
.\runscript.bat

Check GPU:

nvidia-smi

Test Ports:
