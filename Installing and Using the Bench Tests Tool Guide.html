Installation Guide

Follow these steps to install the Bench Tests Tool

  1. Install the Vitrea software as described in the applicable installation guide.
  2. Navigate to the Vitrea_Resources\Vitrea installer Bundle folder location.
  3. Run the BenchTest.msi file.
  4. After installation, access the tool from within Vitrea software.
  1. Sign in as an administrator.
  2. Create a Vitrea_Resources folder on the data drive.
  3. Copy the Vitrea software bundle into the Vitrea_Resources folder.
  4. Navigate to the Vitrea_Resources\Vitrea Installer Bundle folder location.
  5. Run the BenchTest.msi file.
  6. Install the VC2022 redistributable package.
  7. Run the VitreaSystemInfoProvisionServices.msi file.

Usage Instructions

How to run and interpret the Bench Tests Tool

  1. Double-click the Start Vitrea icon on the desktop.
  2. From within Vitrea, select the double-gear icon in the upper right corner.
  3. Select the Run Bench Tests option.
  4. Click OK to confirm when prompted.
  5. Wait for the tests to complete (may take several minutes).
  6. View results in the System Health section under Computer Information.
  1. Navigate to C:\Program Files\Vital Images\benchtest\benchtesttool\benchtesttool.exe.
  2. Run the executable and wait for completion.
  3. View results via the System Checker tool.
  4. Open the SysChkrReport_{machinename,date,time}.html file in any browser.
