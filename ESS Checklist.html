Resolution:
  • Verify inbound firewall rules for ports 3002 (VitreaReceive) and 3003 (VitreaR)
  • Enable ICMPv4-In for Echo Request in Windows Firewall:
    • Open Control Panel > Windows Firewall > Advanced Settings > Inbound Rules
    • Enable "File and Printer Sharing (Echo Request - ICMPv4-In)"
  • Confirm correct IP address and port in Archive configuration
  • Ensure querying device's IP is within allowed scope for port 3003

  • Disable hardware-accelerated decoding:
    • Open Remote Desktop Connection > Experience tab
    • Uncheck "Enable hardware-accelerated decoding"
  • Ensure all monitors have identical settings:
    • Resolution: Minimum 1280x1024, Recommended 1920x1200, Maximum 3840x2160 for VSTP apps
    • DPI: Match across all monitors
    • Orientation: Consistent (all landscape or portrait)
  • Verify Remote Desktop license server and mode configuration
  • Confirm Drives and Clipboard are enabled in RD Session Host settings

Resolution:
  • Ensure 120GB free space on E: drive (100GB for page file, 20GB for AutoDelete)
  • Set virtual memory to 1000000 MB for both Initial and Maximum size on E: drive
  • Configure in Regedit:
    • HKEY_LOCAL_MACHINE\SOFTWARE\Vital Images\Vitrea\VitreaAvSlotSize to "large"
    • HKEY_LOCAL_MACHINE\SOFTWARE\Vital Images\VIMS\ProductDCM to "false"
  • Reboot and verify Session Stats (Available: 2 / Capacity: 3 / Passive: 0)
  • Use NVIDIA RTX 4000 Ada 20GB graphics cards for 1K support
Resolution:
